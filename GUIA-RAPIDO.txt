🚀 COMO VISUALIZAR O SITE
═══════════════════════════════════════════════════════════════

OPÇÃO 1 - Abrir Diretamente:
   1. Navegue até a pasta do projeto
   2. Clique duas vezes em "index.html"
   3. O site abrirá no seu navegador padrão

OPÇÃO 2 - Servidor Local (Recomendado):
   1. Abra o terminal/prompt na pasta do projeto
   2. Execute: python -m http.server 8000
   3. Abra: http://localhost:8000

🖼️ ADICIONAR LOGOS DE PARCEIROS
═══════════════════════════════════════════════════════════════

1. Coloque as imagens na pasta "parceiros/"
2. Nomeie como: partner1.png, partner2.png, etc.
3. Formato recomendado: PNG (300x150 pixels)
4. Os logos aparecerão automaticamente no carrossel

📝 PERSONALIZAR CONTEÚDO
═══════════════════════════════════════════════════════════════

TEXTOS:
   • Edite os arquivos .html com qualquer editor de texto
   • Procure pelo texto que deseja alterar
   • Salve e recarregue a página

CORES:
   • Abra: css/style.css
   • Procure por ":root" no início do arquivo
   • Altere as variáveis de cor
   • Exemplo: --primary-color: #003580;

IMAGENS:
   • Substitua os arquivos existentes mantendo o mesmo nome
   • Ou atualize o caminho no HTML

INFORMAÇÕES DE CONTATO:
   • Edite: contato.html
   • Atualize telefone, email e endereço
   • Modifique horário de atendimento
