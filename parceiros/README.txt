PASTA PARCEIROS - INSTRUÇÕES
=============================

Esta pasta deve conter os logotipos dos parceiros da Jortek.

FORMATO DOS ARQUIVOS:
- Formatos aceitos: PNG, JPG, JPEG, SVG
- Nomes sugeridos: partner1.png, partner2.png, partner3.png, etc.
- Dimensões recomendadas: 300x150 pixels (proporção 2:1)
- Fundo: Preferencialmente transparente (PNG)

COMO ADICIONAR LOGOS:
1. Adicione as imagens dos parceiros nesta pasta
2. Nomeie os arquivos como: partner1.png, partner2.png, partner3.png, etc.
3. Os logos aparecerão automaticamente no carrossel da página inicial

OBSERVAÇÃO:
O carrossel está configurado para exibir até 6 logos de parceiros.
Se você adicionar mais ou menos logos, o JavaScript ajustará automaticamente.
